HealthDataHub: Leveraging Big Data for Healthcare Optimization

  • Client Taylor's University
  • Category Data Science
  • Date February, 2024

For our project on implementing big data technologies in Electronic Health Records (EHRs), collaborating closely with the team to analyze and optimize data processing workflows. My main focus was on leveraging Apache Spark for real-time data analytics and machine learning applications within the healthcare domain. Using Python and Spark's MLib library, I developed predictive models to identify patterns and trends in EHR data, aiming to improve patient outcomes and streamline clinical decision-making processes. Additionally, I contributed to the design and implementation of scalable data processing systems, ensuring efficient handling of large volumes of EHR data. This project provided me with valuable hands-on experience in applying big data technologies to healthcare, furthering my skills in data analysis and contributing to advancements in medical data management.

In my role, enabling flexible and real-time access to comprehensive patient records. This involved designing data schemas, optimizing queries, and ensuring data consistency and security. Moreover, I actively participated in performance tuning exercises, fine-tuning Spark jobs and optimizing resource utilization to enhance overall system efficiency. Through this project, I gained practical insights into the complexities of managing and analyzing healthcare data, along with valuable experience in utilizing cutting-edge big data technologies to address real-world challenges in the healthcare sector.
