Bid4u: Online Auction System

  • Client Taylor's College
  • Category Web Development
  • Date August, 2021

For my Online Auction System (OAS) project, this project was a culmination of my efforts during my tenure at Taylor's College, where I applied my skills in web development and database management to create a robust and user-friendly system. The OAS project was primarily built using PHP and MySQL, providing a solid foundation for backend logic and data storage. Additionally, HTML, CSS, and JavaScript were employed to craft the frontend interface, ensuring an intuitive and visually appealing user experience.

Key features of the OAS project include:

1. User Authentication: Users can register as either admins or buyers, each with their own set of privileges and access levels. Login credentials are securely stored and managed within the database.
2. Item Management: Admins have the capability to add, edit, and delete auction items through a dedicated admin portal. This includes uploading item details such as name, description, starting bid, and images.
3. Auction Bidding: Buyers can browse through available auction items, place bids, and monitor bidding activity in real-time. The system automatically updates the current highest bid and notifies users of bid status changes.
4. Livestream Integration: An innovative feature of the OAS project is the integration of livestream functionality for selected auction events. This enhances the interactive nature of the platform, allowing users to participate in auctions remotely.
5. Secure Transactions: Payment processing and transaction management are handled securely within the system, safeguarding user information and financial data.
6. Admin Dashboard: Admins have access to a centralized dashboard where they can view key metrics, manage user accounts, and monitor auction activity. This provides administrators with the tools they need to effectively oversee the entire auction process.
